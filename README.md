# Professional Telegram Course Bot

A highly modular and dynamic Telegram bot to manage files, courses, hacks, and materials natively via Telegram File IDs.

## Features
- **Whitelist Access**: Only approved users can view files.
- **Role Control**: Owner, Admins, and Users.
- **Dynamic Categories**: Fully flexible categorization (Courses, Hacks, Resources, etc.). Admins can map files to any category natively without hardcoded options.
- **Smart Material Recognition**: Automatically catches forwarded files, links, videos, zips, PDFs, photos and sends them exactly as formatted when requested.
- **Native Broadcasting & Analytics System**.
